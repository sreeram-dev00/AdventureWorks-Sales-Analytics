# AdventureWorks Sales Analytics

## Overview
This project performs a complete sales analysis on the AdventureWorks 
dataset — a simulated bicycle manufacturing and retail company operating 
across multiple countries from 2010 to 2014.

The analysis covers 60,398 sales transactions across 5 years, 
18 countries, and 4 product categories, delivering insights on 
revenue trends, profitability, customer behaviour, and regional performance.

## Tools Used
- Microsoft Excel  — Data cleaning, modeling, pivot tables, dashboards
- SQL (MySQL)      — Querying, aggregations, joins, window functions
- Tableau          — Interactive charts and visual dashboards

## Dataset
- Source : AdventureWorks Sample Database (Microsoft)
- Tables : FactInternetSales, DimCustomer, DimProduct, 
           DimProductCategory, DimProductSubCategory,
           DimSalesTerritory, DimDate
- Total Rows : 60,398 transactions

## Key Business Questions Answered
1. What is the year-wise and month-wise revenue trend?
2. Which products, customers and regions drive the most revenue?
3. What is the overall profit margin and how has it changed over time?
4. How does sales performance vary by quarter and financial year?
5. Who are the top 20 customers and top 20 products by revenue?

## Key Findings
- Total Revenue    : $29.3 Million across 5 years
- Total Profit     : $12.08 Million
- Profit Margin    : 41.15%
- Best Year        : 2013 — revenue jumped to $16.3M (180% growth over 2012)
- Top Region       : Australia — contributing 31% of global revenue
- Top Category     : Bikes — accounting for 96% of total revenue
- Top Product      : Mountain-200 Black, 46
- Top Customer     : Nichole Nara

## Project Structure
- Excel Workbook   — 15 sheets including Sales Data, KPI Analysis,
                     Year/Month/Quarter breakdowns, Pivot Table,
                     Product/Customer/Region Performance, Dashboard
- SQL Queries      — All analytical queries mapped to project questions
- Tableau          — Bar, Line, Pie, Combo charts and interactive dashboard

## Skills Demonstrated
- Data Modeling (Star Schema with 6 related tables)
- DAX-style calculated fields (Sales Amount, Production Cost, Profit)
- Date intelligence (Financial Year, Financial Quarter, Weekday analysis)
- Window functions and aggregations in SQL
